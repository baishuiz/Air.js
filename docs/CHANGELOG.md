# 变更日志（CHANGELOG）

本文档记录 Air.js 的所有版本更新、新增功能、改进和修复。

遵循 [Semantic Versioning](https://semver.org/) 规范。

> **版本格式**：MAJOR.MINOR.PATCH
> - **MAJOR** — 不兼容的 API 变更
> - **MINOR** — 向后兼容的功能新增
> - **PATCH** — 向后兼容的 bug 修复

## 升级指南

### 从 0.1.x 升级至 0.2.0

不需要更改任何代码！0.2.0 完全向后兼容 0.1.x。

**建议升级理由**：
- 性能提升 30%（依赖项解析优化）
- 更好的错误提示和调试支持
- 完整的 TypeScript 类型定义（如果需要）

**升级步骤**：
1. 更新 package.json 中的版本号
2. 运行 `npm install`
3. 运行 `npm test` 验证现有代码兼容性
4. 部署新版本

### 从 0.1.0 升级至 0.1.11

**变更内容**：
- API 保持不变
- 改进构建脚本和测试配置
- 文档增强

**升级步骤**：
同上

## 已知问题

### 0.1.11
- 某些旧版浏览器（IE 8 及更早版本）可能不支持
- 大型项目（>1000 个模块）性能可能下降

### 解决方案
- 使用现代浏览器或 polyfills
- 将大型项目分割成多个构建包
